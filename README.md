# Proyecto-Taller-de-Bases-de-Datos-INEGI2020
Proyecto para Taller de Bases de Datos

Este proyecto consiste en el diseño, implementación y explotación de una base de datos relacional construida a partir de datos oficiales del INEGI (Instituto Nacional de Estadística y Geografía). El objetivo principal es que permita consultas eficientes, análisis estadístico y práctica real de administración de bases de datos.

El proyecto fue desarrollado con un enfoque académico, como parte de la formación en Ingeniería en Sistemas Computacionales, aplicando conceptos de modelado relacional, normalización.
